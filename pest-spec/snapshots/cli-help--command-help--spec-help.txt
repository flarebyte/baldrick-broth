Usage: baldrick-broth test [options] [command]

Test the library for defects

Options:
  -h, --help      display help for command

Commands:
  unit            Run node.js unit tests
  pest            Run acceptance tests for the CLI
  pest1           Run one acceptance tests for the CLI
  scc             Count lines of code
  cli             Run client directly
  help [command]  display help for command
